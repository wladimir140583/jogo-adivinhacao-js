3. Entrada e Saída

3.1 printf - Saída de Dados

Exibe informações na tela (console).

printf("Texto");
printf("Numero: %d", variavel);
printf("Nome: %s, Idade: %d", nome, idade);

3.2 Especificadores de Formato

Especificador Tipo Exemplo
%d ou %i int printf("%d", 10);
%f float / double printf("%f", 3.14);
%.1f float com 1 decimal printf("%.1f", 3.14);
%c char printf("%c", 'A');
%s string printf("%s", "texto");
%p ponteiro printf("%p", &var);
Exemplos de printf: 
int idade = 25;
float nota = 8.75;
char letra = 'A';

printf("Idade: %d\n", idade);           // Idade: 25
printf("Nota: %.2f\n", nota);           // Nota: 8.75
printf("Letra: %c\n", letra);           // Letra: A
printf("Idade: %d, Nota: %.1f\n", idade, nota);

3.3 Caracteres Especiais

Código Significado
\n Nova linha
\t Tabulação
\\ Barra invertida
\" Aspas duplas
\' Aspas simples

3.4 scanf - Entrada de Dados

Lê dados digitados pelo usuário.

int numero;
scanf("%d", &numero);  // & = endereço da variável
CRÍTICO: Sempre use o operador & antes da variável no scanf (exceto para strings). Ele indica o endereço de memória onde o valor será armazenado.
Exemplos de scanf: 
int idade;
float altura;
char letra;

printf("Digite sua idade: ");
scanf("%d", &idade);

printf("Digite sua altura: ");
scanf("%f", &altura);

printf("Digite uma letra: ");
scanf(" %c", &letra);  // Espaço antes do %c

3.5 Lendo Strings

Método 1: scanf (sem espaços)

char nome[50];
scanf("%s", nome);  // Lê até o espaço (sem &)

Método 2: scanf com formatação (com espaços)

char nome[50];
scanf(" %[^\n]", nome);  // Lê até o Enter

Método 3: fgets (mais seguro)

char nome[50];
fgets(nome, 50, stdin);
nome[strcspn(nome, "\n")] = '\0';  // Remove \n
DIFERENÇA:
  • scanf("%s") para uma palavra
  • scanf(" %[^\n]") ou fgets() para texto com espaços
  • fgets captura o Enter, precisa remover com strcspn

4. Operadores

4.1 Operadores Aritméticos

Operador Operação Exemplo
+ Adição a + b

* Multiplicação a * b
/ Divisão a / b
% Módulo (resto) a % b
Exemplos: 
int a = 10, b = 3;

float x = 10.0, y = 3.0;
float divisao = x / y;   // 3.333...
ATENÇÃO - Divisão Inteira: Quando ambos operandos são int, o resultado é int. 10 / 3 = 3 (não 3.33). Use float para resultado decimal.

5. Estruturas Condicionais

5.1 if (SE)

if (condicao) {
    // código executado se verdadeiro
}
Exemplo: 
int idade = 20;

if (idade >= 18) {
    printf("Maior de idade\n");
}

5.2 if...else (SE...SENÃO)

if (condicao) {
    // código se verdadeiro
} else {
    // código se falso
}
Exemplo: 
int nota = 6;

if (nota >= 7) {
    printf("Aprovado\n");
} else {
    printf("Reprovado\n");
}

5.3 if...else if...else

if (condicao1) {
    // código se condicao1 verdadeira
} else if (condicao2) {
    // código se condicao2 verdadeira
} else {
    // código se todas falsas
}
Exemplo - Sistema de Notas: 
float media = 8.5;

if (media >= 7.0) {
    printf("Aprovado\n");
} else if (media >= 4.0) {
    printf("Recuperação\n");
} else {
    printf("Reprovado\n");
}

5.4 Operador Ternário

Forma compacta de if-else simples:

variavel = (condicao) ? valor_se_verdadeiro : valor_se_falso;
Exemplo: 
int idade = 20;
char *status = (idade >= 18) ? "Maior" : "Menor";
printf("%s de idade\n", status);

5.5 switch...case

Para múltiplas opções baseadas em um valor:

switch (variavel) {
    case valor1:
        // código
        break;
    case valor2:
        // código
        break;
    default:
        // código padrão
}
Exemplo: 
int opcao = 2;

switch (opcao) {
    case 1:
        printf("Opção 1\n");
        break;
    case 2:
        printf("Opção 2\n");
        break;
    case 3:
        printf("Opção 3\n");
        break;
    default:
        printf("Opção inválida\n");
}
IMPORTANTE: Não esqueça o break em cada case! Sem ele, a execução continua para o próximo case.

7. Strings

7.1 Declaração e Uso

Strings em C são arrays de caracteres terminados com '\0'.

char nome[50];              // Declara string
char saudacao[] = "Olá";   // Inicializa com texto
char letra = nome[0];       // Acessa primeiro caractere

7.2 Funções de String (string.h)

Função Descrição Exemplo
strlen() Tamanho da string strlen("Olá") = 3
strcpy() Copia string strcpy(dest, origem)
strcat() Concatena strings strcat(str1, str2)
strcmp() Compara strings strcmp(str1, str2)
strcspn() Encontra caractere strcspn(str, "\n")
Exemplo de uso: 
#include <string.h>

char nome1[50] = "João";
char nome2[50];

strcpy(nome2, nome1);         // Copia "João" para nome2
int tam = strlen(nome1);      // tam = 4
strcat(nome1, " Silva");      // nome1 = "João Silva"

9. Erros Comuns

9.1 Esquecer o ponto e vírgula

Errado: 
int x = 10
printf("Valor: %d", x)
Correto: 
int x = 10;
printf("Valor: %d", x);

9.2 Esquecer o & no scanf

Errado: 
int idade;
scanf("%d", idade);
Correto: 
int idade;
scanf("%d", &idade);

9.3 Usar %d para float

Errado: 
float nota = 8.5;
scanf("%d", ¬a);   // Usa %f, não %d!
Correto: 
float nota;
scanf("%f", ¬a);

9.4 Esquecer #include

Errado: 
// Sem #include
int main() {
    printf("Olá");  // ERRO: printf não declarado
}
Correto: 
#include <stdio.h>

int main() {
    printf("Olá");
    return 0;
}

9.5 Divisão inteira inesperada

Problema: 
int a = 10, b = 3;
float resultado = a / b;  // resultado = 3.0 (não 3.33)
Solução: 
int a = 10, b = 3;
float resultado = (float)a / b;  // resultado = 3.33
